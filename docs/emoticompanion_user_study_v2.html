统一主试脚本

开场：“谢谢参加。你会实际使用一次这个音乐陪伴系统。你将由自己点击开始和结束，录制一段短视频；系统会分析视频中的面部表情、语音情绪与疲劳状态，再生成音乐。原始视频默认不会作为研究数据长期保存，你可以随时停止。”

基线：“在看到系统结果前，请先按真实感受回答当前状态。这里没有正确答案。”

视频录制：“请在系统页面点击‘开始录制’，自然说一段你此刻的想法，并让面部保持在画面中；说完后由你自己点击‘结束录制’。”

音乐 A/B：“请独立评价每段音乐。两段来自不同系统策略，不代表其中一段一定更好。”

收尾：“最后请评价这次视频交互、隐私、等待时间，以及你最喜欢和最希望改进的部分。”

不要在 A/B 评分前告诉参与者哪一段是 ToM 或 Standard，也不要展示 CoT、need 或 music spec，以免产生期待偏差。
