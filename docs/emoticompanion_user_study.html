统一主试脚本

开场：“谢谢参加。你会实际用一下这个音乐陪伴系统：对着摄像头说一句你现在真实的想法，它会判断你的情绪，并生成一段音乐。原始脸部图像和录音默认不会保存，只记录匿名标签和你的评分。你可以随时停止。”

诱导：“先看一小段视频，帮助你进入一种状态。看完后请按真实感受回答，不需要迎合目标情绪。”

系统输入：“现在请拍一张照片，再录一句你此刻真实的想法。我来运行系统。”

音乐 A/B：“请独立评价每段音乐。两段只是不同系统版本，不代表一段一定更好。”

收尾：“最后请评价隐私、等待时间，以及你最喜欢和最希望改进的部分。”

不要在 A/B 评分前告诉参与者哪一段是 ToM 或 Standard，也不要展示 CoT/need/music spec，以免产生期待偏差。
